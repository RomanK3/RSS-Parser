## RSS-Parser

RSS-Parser is a MERN which was built with using [RSS Feed](https://lifehacker.com/rss).

## Instalation

Clone or download this repositoty.

Open folder src:

``` cd src ```

Back-end: 

``` cd back-end ```

Install the dependencies:

``` npm i ```

Then you can start the Back-end part:

``` npm start dev ```

Front-end: 

``` cd front-end ```

Install the dependencies:

``` npm i ```

Then you can start the Back-end part:

``` npm start ```
